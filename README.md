CSCI 32913 Fall 2021 Final Project

I created an online multiplayer Uno game that is actually scalable in the since that it can support multiple games occuring at once.
Hosted on Heroku.
I wrote my own packet protocal, albeit a simple one. I also made a stand-alone GUI for users in the form of a jar file.
